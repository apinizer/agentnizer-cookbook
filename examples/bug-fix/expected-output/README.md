## Why we trimmed the rest

[`../../quickstart/expected-output/`](../../quickstart/expected-output/) —
five reviews, tests, qa, docs, retro, handoffs. That structure is the
same regardless of recipe; what changes for bug-fix is **the upstream
discipline**, so we focus the example on those three files.
